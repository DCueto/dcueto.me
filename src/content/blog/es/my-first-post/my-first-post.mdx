# Este es el primer post del blog

Astro 2.0 ha llegado con una *plétora de nuevas características y mejoras* que hacen que la construcción de sitios web modernos sea aún más agradable. En este post, exploraremos algunas de las actualizaciones más emocionantes, incluyendo un rendimiento mejorado, nuevas integraciones y una mejor experiencia para los desarrolladores.

## Rendimiento Mejorado

Uno de los principales **aspectos destacados** de **Astro 2.0** es su *rendimiento mejorado*. La nueva versión introduce optimizaciones que reducen significativamente los tiempos de construcción y mejoran el rendimiento en tiempo de ejecución. Esto significa sitios web más rápidos y una experiencia de desarrollo más fluida.

Lista de mejoras de rendimiento:

1. Tiempos de construcción más rápidos
2. Mejor rendimiento en tiempo de ejecución
3. Tamaños de bundle reducidos

## Experiencia de Desarrollador Mejorada

La experiencia de desarrollador ha sido un foco importante en Astro 2.0. La nueva versión incluye mejores mensajes de error, documentación mejorada y una API más intuitiva. Estos cambios hacen que sea más fácil para los desarrolladores comenzar con Astro y construir sitios web increíbles.

const dateOptions: Intl.DateTimeFormatOptions = { day: 'numeric', month: 'short', year: 'numeric' };

export async function getStaticPaths() {
  const blogPosts = await getCollection("blog") as CollectionEntry<'blog'>[];

  const paths = blogPosts.map((blogPost) => {
    const url = `${blogPost.slug}`;
    const locale = blogPost.data.language;

    return {
      params: {
        locale,
        path: url,
      },
      props: {
        blogPost
      }
    }
  });

  return paths;
}
